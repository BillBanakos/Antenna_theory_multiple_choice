# Κουίζ Θεωρίας Κεραιών (Antenna Theory Quiz App) 📡

Μια σύγχρονη desktop εφαρμογή γραμμένη σε **Python** και **PyQt6** για την εξάσκηση στο μάθημα της **Θεωρίας Κεραιών** (βασισμένη στο σύγγραμμα του Balanis).  

Η εφαρμογή προσφέρει ερωτήσεις πολλαπλής επιλογής, με πλήρη υποστήριξη δύο γλωσσών (**Ελληνικά και Αγγλικά**) και σχεδιάστηκε για να βοηθήσει τους φοιτητές στην κατανόηση βασικών και προχωρημένων εννοιών της θεωρίας κεραιών.

---

## ✨ Χαρακτηριστικά

### 🌐 Δίγλωσσο Περιβάλλον (Bilingual)

- Άμεση εναλλαγή μεταξύ **Ελληνικών** και **Αγγλικών**.
- Η αλλαγή γλώσσας επηρεάζει:
  - Το γραφικό περιβάλλον (UI).
  - Το κείμενο των ερωτήσεων.
  - Τις απαντήσεις και τα αποτελέσματα.
- Η εναλλαγή γίνεται σε πραγματικό χρόνο με ένα κουμπί στο πάνω δεξιά μέρος της εφαρμογής.

---

Η εφαρμογή περιλαμβάνει ερωτήσεις από επιλεγμένα κεφάλαια της Θεωρίας Κεραιών:

| Κεφάλαιο | Περιγραφή |
|----------|-----------|

# 🎯 Λειτουργίες Κουίζ (Quiz Modes)

Η εφαρμογή διαθέτει τρεις διαφορετικούς τρόπους εξάσκησης:

## 1. Με τη σειρά

- Οι ερωτήσεις του επιλεγμένου κεφαλαίου εμφανίζονται με τη σειρά που υπάρχουν στη βάση δεδομένων.
- Κατάλληλο για συστηματική μελέτη ενός κεφαλαίου.

## 2. Ανακατεμένα

- Οι ερωτήσεις του επιλεγμένου κεφαλαίου εμφανίζονται σε τυχαία σειρά.
- Βοηθά στην καλύτερη αξιολόγηση της πραγματικής κατανόησης.

## 3. Τεστ (50 Ερωτήσεις)

- Προσομοίωση εξέτασης με έως **50 τυχαίες ερωτήσεις**.
- Επιλογή:
  - Ενός συγκεκριμένου κεφαλαίου.
  - Όλων των διαθέσιμων κεφαλαίων.

# 📂 Δομή Κώδικα

Το project είναι σχεδιασμένο ως **standalone εφαρμογή**.

Όλη η λειτουργικότητα βρίσκεται σε ένα αρχείο `.py`:

- Γραφικό Περιβάλλον (GUI).
- Λογική εφαρμογής.
- Διαχείριση κουίζ.
- Εναλλαγή γλωσσών.
- Αποτελέσματα.
- Βάσεις ερωτήσεων.

Η εφαρμογή περιλαμβάνει τις λίστες:

```python
ALL_QUESTIONS_GR
```

για τις ελληνικές ερωτήσεις

και:

```python
ALL_QUESTIONS_EN
```

για τις αγγλικές ερωτήσεις.

Η συγκεκριμένη σχεδίαση κάνει την εφαρμογή:

- Εύκολη στη μεταφορά.
- Απλή στην εγκατάσταση.
- Κατάλληλη για χρήση χωρίς εξωτερικές βάσεις δεδομένων.
